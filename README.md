# Проект: Путешествие по России

**Проект доступен по ссылке на** [GitHub Pages]()

### Описание

Проект представляет собой адаптивный сайт, посвященный путешествиям по России.

### Использованные технологии:

- Адаптивная верстка;
- flex- и grid-контейнеры;
- псевдоклассы;
- медиазапросы;
- методология БЭМ.
